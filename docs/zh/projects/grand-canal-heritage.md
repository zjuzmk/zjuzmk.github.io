# 大运河遗产保护

**2015–2016** · 硕士论文 · 香港大学

## 概述

本项目源自我在香港大学的硕士论文，探讨官方遗产框架与中国大运河——全长逾1700公里的联合国教科文组织世界遗产——保护实践中的错位。以杭州段为案例，研究揭示了"权威遗产话语"与运河沿岸城市社区居民日常体验之间的张力。

## 研究问题

- 权威遗产话语如何形塑中国大运河保护政策与实践？
- 当自上而下的遗产框架遭遇地方城市发展压力时，会发生什么？
- 谁的遗产被保护——又为谁而保护？

## 核心发现

杭州案例揭示了一种根本性的错位：官方遗产话语强调运河的物质真实性与历史纪念碑性，而当地社区则将运河视为嵌入日常生活的生活化基础设施。这种断裂产生的保护成果，虽满足全球遗产标准，却疏离了当地居民。
