# 北京2022冬奥会文化遗产

**2022–2024** · SNSF研究项目 · EPFL

## 概述

作为瑞士国家科学基金会（SNSF）资助的国际研究项目"北京2022年冬奥会文化遗产的利用"的一部分，本研究探讨中国首届冬奥会如何将文化遗产——包括工业与非物质的——用于城市更新与国家品牌建设。

在一个横跨地理学、建筑学、人类学与社会学的跨学科团队中，项目考察了遗产在奥运盛事的宏大叙事中被定义、保存与再利用的创造性——有时是矛盾性的——方式。

## 研究焦点

- 中国政府如何采取一种**国家主导的工业遗产方法**，区别于西方市场驱动或公民社会主导的模式？
- 在大型赛事背景下，遗产在**资本积累与城市更新**中扮演什么角色？
- 不同层面的遗产叙事——地方的、国家的、全球的——如何在奥运驱动的发展中被协商？

## 核心发现

北京2022案例揭示了一种独特的国家主导模式：工业遗产保护同时服务于资本积累、城市更新与遗产保存——这三者在其他情境中常常面临张力。
